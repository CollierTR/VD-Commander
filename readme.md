# VD Commander

This is a simple CLI wrapper that simplifies pyvda for use with keybinding applications such as AutoHotKey. 

You can have your keys run the script contained in main.py and pass in arguments depending on what you want the key's action to perform. I have mine set up like this:

Meta + (1-6) moves to workspace 1-6
alt + (1-6) moves active window to workspace 1-6
Meta + alt + (1-6) moves active window to workspace 1-6 and changes to that workspace (follows the window.)
Meta + P pins/unpins the active window to all workspaces
